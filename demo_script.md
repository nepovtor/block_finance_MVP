# Block Finance MVP Demo Script

## Opening

Young users already spend money digitally, but banking apps rarely make that behavior feel rewarding or engaging.

## Product idea

Block Finance turns everyday payments into instant game rewards that drive progress and repeat engagement.

## Live demo steps

1. Open the app and land on the onboarding screen.
2. Move to the dashboard and point out the user profile, XP, streak, and reward slot.
3. Click `Pay for coffee` and explain that this simulates a real banking action hitting the backend.
4. Show the reward card lighting up with `extra_move`.
5. Open the game and point out that the same reward is visible in the game UI.
6. Place a few pieces, clear a line, and let the score rise.
7. Reach a dead-end state, then use the reward to continue the run once.
8. Finish with game over or bank the score and restart.

## Closing value

That is the whole loop: financial behavior creates instant motivation, motivation powers gameplay, and gameplay feeds progress that can support loyalty and retention.
